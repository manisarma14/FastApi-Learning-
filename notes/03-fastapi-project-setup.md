# FastAPI Project Setup

## Environment Setup

Create a project folder and virtual environment.

Benefits:

* Dependency isolation
* Cleaner project structure
* Easier deployment

## Install FastAPI

Install:

* FastAPI
* Uvicorn

## Why Uvicorn?

Uvicorn is an ASGI server that runs FastAPI applications.

## Development Workflow

Write code → Run server → Test API → Make changes → Restart or reload

## Key Learning

A virtual environment ensures project dependencies remain separate from other projects.
