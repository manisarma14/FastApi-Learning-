# Path Parameters

## What are Path Parameters?

Path parameters identify a specific resource.

Example:

/users/5

Here 5 represents a specific user.

## Common Examples

/users/10

/products/25

/bookings/7

## Characteristics

* Usually required
* Identify a specific record
* Part of the URL path

## FastAPI Advantage

FastAPI automatically validates parameter types.

Invalid values result in automatic validation errors.

## Key Learning

Path parameters answer the question:

"Which specific resource?"
