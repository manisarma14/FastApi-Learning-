# FastAPI Routing

## What is a Route?

A route connects a URL to a specific function.

Example:

/users

may return all users.

## Why Routes Matter

Routes allow clients to access different backend functionalities.

Examples:

/users

/products

/bookings

## REST API Design

GET /users

POST /users

GET /users/{id}

PUT /users/{id}

DELETE /users/{id}

## Key Learning

Routes act as entry points into the backend application.
