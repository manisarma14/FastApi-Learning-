# Query Parameters

## What are Query Parameters?

Query parameters filter, search, sort, or paginate data.

Example:

/users?page=2

/products?category=laptop

## Characteristics

* Usually optional
* Used for filtering and searching
* Appear after ?

## Common Use Cases

Pagination

Searching

Sorting

Filtering

## Path vs Query

Path Parameter:

Identifies a specific resource

Query Parameter:

Controls how data is returned

## Key Learning

Query parameters answer the question:

"How should data be returned?"
